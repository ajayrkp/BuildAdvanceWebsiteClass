    6. img - specifies images -- It's a self quote element; Parameters are:

    • src - specify the path of image file
    • alt - Alt text -- text when image is not shown
    • title - Text when hover on image
    • width, height - width, height to be shown on web pages
    • align - Align the image left, right, top, middle, bottom
